# Automação de Testes com Selenium e Java
Este é o repositório dos Testes Automatizados em Java e Selenium para o Projeto Leilão da Alura.

## Como executar o projeto

### Pré-requisitos:
- Java instalado (JDK 8 ou superior)
- Eclipse ou outra IDE Java instalada
- Download do projeto em sua máquina
- Maven e JUnit configurados

### Instalando as dependências do Maven

Para instalar as dependências do Maven, utilize o atalho:

```
ALT+F5
```

### Execução local (em sua máquina)

Para executar o Projeto Leilão localmente, utilize o seguinte procedimento:
1. Suba o Projeto Leilão localmente (execute como Java Application):

```
LeilãoApplication.java
```

Ele estará acessível localmente na porta:

```
http://localhost:8080/
```

2. Execute os testes JUnit na pasta:

```
src/test/java
```
